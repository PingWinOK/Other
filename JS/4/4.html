Типовые задачи обработки одномерных числовых массивов

1.Создать и вывести массив, состоящий из n элементов целого типа. Произвести следующие манипуляции, каждый раз выводя результат:





1) Удалить из массива элементы со 2-го по 4-й.

2) Добавить 2 элемента в начало массива.

3) Изменить порядок элементов массива на противоположный.

4) Отсортировать элементы массива по убыванию.

2.Создать и вывести массив, состоящий из n элементов целого типа. Найти количество элементов четных по значению.

4.Создать и вывести массив целочисленный массив, состоящий из n элементов. Поменять местами максимальный и минимальный элементы.
